  • Tenovar Ltd.

    2006

    Voice Response System for Auto Credit Card Payment

    Developer

    The project for CVA Malta was to monitor all vehicles entering and exiting the capital city Valletta and charged them on the parking time within the city. We provided them with Avaya IP400 system for their call center. They would like to have a voice response system that can help car owners pay bills via telephone.

    I was responsible for making such integration between customer's database system and Avaya IPO system using Voicemail Pro. I implemented the entire solution using the Avaya Call Flow tool. The challenge was I could not use the built in text-to-speech engine as lack of Maltese support. I solved this issue by using very limited components provided by the call flow utility. I recorded some Maltese sentences and words, used VB script to split the car number plate value as well as credit card number and used audio play components and condition components to simulate the text to speech feature. In order to improve the performance I used binary search algorithm on characters recognition to smooth the reading speed and avoided the components invoking limits of the system. I also did a training session for technician from CVA in the end of the project.

    MySQL, Avaya IP Office, Voicemail Pro
